##电器商城
	一.前端用户系统
		1.用户的登录与注册，使用邮箱接收验证码进行注册验证。
		2.商品的分页显示，多条件查询，关键词查询。
		3.购物车模块，商品加入购物车后可在购物车界面进行增删改以及支付操作。
		4.支付模块，接入了支付宝的沙箱支付模拟真实的业务流程。
		5.个人中心:
			a.用户的头像等相关信息进行修改；
			b.收货地址管理，接入了百度地图实现对收货地址的定位以及位置抓取。
			c.订单管理，查看订单状态，对生成的订单进行支付操作，退款等业务功能还有待完善
	二.后台管理系统
		1.用户登录，自动识别用户身份，分配不同的权限 （分为管理员和超级管理员，超级管理员登录后可对管理员进行管理）。
		2.用户信息管理，管理员信息管理。
		3.商品类型和子类型管理。
		4.商品信息管理，库存信息管理。
		5.订单管理。
